# Fireside Recap Narration: Elco / Voice Workbench

## Intro
Fireside recap: Elco on voice-driven workbenches, specialized agents, creative production systems, and access through persistent building.

## Voice-driven workbench
Elco is building a voice-driven workbench where users can speak intent, have the system break work into parts, and dispatch specialized agents.

## Agent roles and handoffs
The agent system includes role-specific personas for engineering, design, concierge, content, trading, testing, and other functions, with shared memory and handoff behavior.

## Creative production system
A tarot deck and design-system project showed how the setup can support complex creative production through research, memory, structured generation, and many iterations.

## Concrete rough edges
The rough edges were concrete: voice reliability, profile serialization, turn-bridge reconnection, subprocess overhead, Discord TTS capture, and safety around misheard destructive commands.

## Access through building
Elco framed the broader opportunity as access. Current AI already lets creative, persistent people build systems that previously required credentials, teams, or institutional permission.
